```python
os.path.isdir() = Confere se uma pasta já existe
os.mkdir() = Cria pastas
os.listdir() = Lista os arquivos de uma pasta
os.rename() = Renomeia o nome(url completa) de um arquivo
```

### Função do python

```python
str.endswith('') = Verifica o final da string de acordo com o parâmetro

"Bruno".endswith("no") # true
"Bruno".endswith("ne") # false

"Bruno".endswith(tuple("no", "ne")) # true
"Bruno".endswith(tuple("na", "ne")) # false
```
