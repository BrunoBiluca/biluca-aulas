
# Replace

```python
txt = "Bruno gato demais"
new_txt = txt.replace("gato", "forte e humilde")
print(new_txt)

# Bruno forte e humilde demais
```

# Split

```python
txt = "welcome,to,the,jungle"  
  
x = txt.split(",")  
  
print(x)

# welcome
# to
# the
# jungle
```
